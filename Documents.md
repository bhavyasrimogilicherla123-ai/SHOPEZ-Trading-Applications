## Project Information

### Project Title

**SHOPEZ Trading Hub – Stock Trading Simulation Platform**

### Technology Stack

**Frontend**

* HTML5
* CSS3
* JavaScript
* Tailwind CSS

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB

### Key Features

* User Registration
* User Login Authentication
* Stock Buy Functionality
* Stock Sell Functionality
* Portfolio Management
* Trading Cash Balance Tracking
* Trade Ledger History
* MongoDB Data Storage
* Responsive User Interface

### Modules

#### Authentication Module

* User Registration
* User Login
* Session Management

#### Trading Module

* Stock Selection
* Buy Orders
* Sell Orders
* Cash Balance Updates

#### Portfolio Module

* Current Holdings
* Average Cost Calculation
* Portfolio Value Tracking

#### Ledger Module

* Transaction History
* Buy/Sell Records
* Timestamp Tracking

### Architecture

```text
Frontend (HTML + Tailwind CSS)
            │
            ▼
Backend (Node.js + Express.js)
            │
            ▼
MongoDB Database
```
